## 用法也很简单
### 备份
- 在Saves里选择一个存档，点击一下即可
- 勾选“I have returned to the main...”
- "Backup Selected Save"

在我的电脑上 3s 即可备份完成

### 恢复

- 在Saves里选择一个存档，点击一下即可
- 在 Backups里选择一个备份
- 点击 "Restore Selected Backup"，随后会弹出一个成功恢复的窗口，10s 后自动关闭

在我的电脑上也是 3s 恢复完成
